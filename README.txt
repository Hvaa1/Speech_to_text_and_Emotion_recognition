1. Chạy bằng file untitled1.ipynb trong folder WhisperFT (tạo bản sao để chạy)
2. Sửa đường dẫn model theo đường dẫn của mình (model whisper tên là whisper_merged, phoBert tên là Phobert_ft1)
3. Sửa !ngrok config add-authtoken YOUR_AUTHTOKEN (link lấy token https://dashboard.ngrok.com/get-started/your-authtoken)
4. Xem tình trạng sever ở sever.log
Note: Đây là bản demo có thể còn sửa chữa ae lấy test thử file tự ghi âm đuôi .wav 
Nếu ấn mà nó không hiện kết quả xem lỗi ở file sever.logS
